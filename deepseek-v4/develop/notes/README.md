# Development Notes

> Ad-hoc investigations, decision records, and troubleshooting notes
> produced during development.

## Naming Convention

Files are named `YYYY-MM-DD-<topic>.md`, e.g.

- `2026-04-30-hc-pp-send-recv.md`
- `2026-05-02-hash-routing-ep.md`
- `2026-05-04-numeric-alignment-with-reference.md`

## Tags (suggested at the top of each note)

- `decision` — architecture / interface decisions
- `troubleshooting` — issues encountered and how they were resolved
- `analysis` — deep dives into the reference implementation
- `experiment` — record of a comparison experiment
- `convergence` — convergence curves / numerical alignment records
